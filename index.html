Preguntas Frecuentes

Respuestas a las preguntas más comunes sobre HorekuOs

¿Cómo puedo añadir HorekuOs a mi grupo?
Para añadir HorekuOs a tu grupo, descarga el script de HorekuOs y alojalo en algún host o simplemente añade el número oficial del bot a tu grupo.
¿Es HorekuOs gratuito?
Sí, HorekuOs es completamente gratuito. Sin embargo, algunos de nuestros códigos estan ofuscasdos, ya sea por los api keys de las APIS u otro motivo.
¿Cuáles son los requisitos para usar HorekuOs?
HorekuOs requiere que tu servidor nodeJS que tenga los recursos básicos, y después simplemente ejecutar el código y poner tu número para obtener el código de vinculación.
¿Puedo personalizar el prefijo de los comandos?
Sí, puedes cambiar el prefijo predeterminado aunque actualmente es multiprefiiopor pero puedes cambiarlo desde "./settings.json".
¿Cómo reporto un error o sugerir una nueva característica?
Puedes reportar errores o sugerir nuevas características a través de nuestro servidor de soporte en WhatsApp o por nuestro correo electrónico: syllscodelab@gmail.co
